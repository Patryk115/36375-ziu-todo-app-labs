# Zaawansowany Interfejs Użytkownika - TodoApp


## 🌿 Struktura repozytorium
* `main` — aktualna, najnowsza i w pełni działająca wersja projektu (obecnie zawiera kod po Lab 5).
* `lab5` — snapshot projektu po zakończeniu **Laboratorium 5** (Dodanie Dashboardu, integracja Material UI oraz Tailwind CSS).
* `lab4` — snapshot projektu po zakończeniu **Laboratorium 4** (Zarządzanie globalnym stanem za pomocą Context API i `useReducer`, obsługa `localStorage`).

---


## 🚀 Jak uruchomić

### Krok 1: Pobranie całego projektu
Sklonuj repozytorium na swój komputer i wejdź do folderu:
```bash
git clone <TUTAJ_WKLEJ_SWÓJ_LINK_DO_REPOZYTORIUM>
cd todo-app-repo
```

### Krok 2: Uruchomienie wybranego laboratorium
```
git checkout <NAZWA_GAŁĘZI>  # np. git checkout lab5
npm install                 
npm run dev
```
